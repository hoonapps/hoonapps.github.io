# Hoonapps

백엔드 개발과 프로젝트 기록을 정리하는 개인 기술 블로그입니다.

기존 영어 글은 그대로 보존하고, 앞으로 새 글은 한국어를 기준으로 작성합니다.
주제는 백엔드, AI 도구, 오픈소스 실험, 프로젝트 기록을 중심으로 정리합니다.

## 다루는 것

- AI 모델, 에이전트, MCP, RAG, 로컬 LLM, 개발 도구
- 백엔드 내부 동작, DB, 트랜잭션, 큐, 캐시, 네트워크, 관측성
- GitHub/Threads/Hacker News에서 발견한 오픈소스 실험
- 새 프로젝트의 설계 결정, 구현 과정, 실패 기록
- 기존 영어 글은 아카이브로 보존

## 방향

- 기존 글은 아카이브로 유지
- 새 글은 한국어 기준으로 작성
- 카테고리별로 읽기 쉽게 정리
- 프로젝트와 실험 기록은 가능한 한 코드와 함께 남기기
